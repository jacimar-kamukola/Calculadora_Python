Esta é uma calculadora desenvolvida em Python, projetada para oferecer uma interface simples e funcional, permitindo que o usuário realize operações matemáticas básicas de forma rápida e intuitiva.

Ela suporta operações básicas como:
1-Adição
2-Subtração
3-Multiplicação
4-Divisão

Tem suporte para numeros decimais.

Código de fácil compreensão, boa para testar habilidades gráficas com python.
